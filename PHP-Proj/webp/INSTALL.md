# WebP优化插件安装指南

## 安装步骤

### 1. 上传插件
将 `webp-optimizer` 文件夹上传到 WordPress 的 `/wp-content/plugins/` 目录下

### 2. 激活插件
在 WordPress 后台 -> 插件 -> 已安装插件 中找到 "WebP Optimizer" 并激活

### 3. 配置设置
进入 设置 -> WebP优化 进行配置

## 推荐配置

```
启用WebP转换: ✓ 启用
WebP质量: 85
转换JPEG图片: ✓ 启用
转换PNG图片: ✓ 启用
保留原图: 不启用（节省空间）
最大宽度: 1920 像素
压缩方式: 自动选择
```

## 功能验证

1. 配置完成后上传一张JPEG或PNG图片
2. 检查上传目录是否生成了对应的WebP文件
3. 查看文件大小对比，WebP应该明显更小

## 故障排除

### 转换不生效
- 检查插件是否已启用
- 确认上传的是JPEG/PNG格式
- 验证系统是否支持WebP

### 转换失败
- 检查服务器内存限制
- 确认上传目录有写入权限
- 查看PHP错误日志

### 质量不理想
- 调整WebP质量参数
- 尝试不同的压缩方式
- 考虑调整图片尺寸限制

## 卸载

1. 停用插件
2. 删除插件文件
3. （可选）删除数据库中的设置：`webp_optimizer_options`
